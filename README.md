# Personal Portfolio Website

## Features

- **About Section:** Detailed overview of skills, tools, and professional experience.  
- **Projects Section:** Interactive project cards with tech stacks and “View Project Case Study” modal.  
- **Case Study Modal:** Full project breakdown with overview, problem, research, design process, solution, and key results.  
- **Dark & Light Theme Support:** Seamless theme switching for optimal viewing experience.  
- **Responsive Design:** Optimized for desktop, tablet, and mobile.
